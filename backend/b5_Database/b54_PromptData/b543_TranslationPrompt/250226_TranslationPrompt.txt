#6 WordListPostprocessing
(역할 및 업무)

(<단어장> 정리방법)
적합: [번역]된 단어가 적합한 경우
삭제: 중요도가 낮은 단어는 해당 단어 자체를 삭제
선택: [번역]에서 가장 적합한 단어 하나만 선택
수정: [번역]된 단어가 적합하지 않은 경우 수정

(<정리된단어장.json>의 구조 및 내용)
{
    '정리된단어장': [
        {
            '번호': n,
            '원문': '[원문]을 그대로 작성',
            '번역': '[번역]에서 가장 적합한 단어 하나만 선택하여 작성',
            '정리방법': '적합, 삭제, 선택, 수정 중 표기',
        },
        {
            '번호': n,
            '원문': '[원문]을 그대로 작성',
            '번역': '[번역]에서 가장 적합한 단어 하나만 선택하여 작성',
            '정리방법': '적합, 삭제, 선택, 수정 중 표기',
        },
        ...
    ]
}

(<정리된단어장.json> 완성 절차)
1. <단어장>에서 [원문]은 [원문언어]를 그대로 작성한 것이고, [번역]은 [원문언어]를 [번역언어]로 번역하여 작성한 것입니다.
2. <단어장>은 단어의[번호]와 [원문] 그리고 그 단어를 [번역언어]로 번역한 [번역] 단어들로 구성되어 있습니다.
3. 첫번째, <단어장>의 [원문]이 [원문내용]을 바탕으로 잘 [번역] 되었을때는 이를 그대로 둡니다. 이 경우는, '번호', '원문', '번역'을 그대로 작성한 뒤, '정리방법'은 '적합'을 표기합니다.
