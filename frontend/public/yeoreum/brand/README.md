Place the STIDIO logo asset here.

- `stidio-logo.png`

Recommended PNG size:
- 320 x 80 px, transparent background
- Keep the actual wordmark inside the image with roughly 16 px horizontal padding
- Export at 2x or 3x if the source is very sharp; the UI displays it at up to about 164 x 41 px

The workspace loads `stidio-logo.png` first and falls back to the text wordmark if the image is missing.
